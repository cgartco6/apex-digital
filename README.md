# Apex Digital – Role‑Based Marketing Suite

A complete, self‑contained web application for design, branding, marketing, and a platform marketing engine that targets 3,000 impulse buyers in 3 days with honest advertising.

## Features
- Role‑based dashboards (Client, Admin, Owner)
- Design Studio (Fabric.js with layers, undo/redo, export)
- Branding Studio (colour/font customisation, ZIP export)
- Marketing Studio (campaign management, CSV/PDF export)
- Direct EFT payment simulation (South Africa)
- Platform Marketing Engine (simulated traffic, no clickbait)
- GDPR / POPIA compliant

## How to Run
1. Clone this repository.
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge).
3. Use the role dropdown to switch between Client, Admin, and Owner.

All data is stored locally in your browser. No backend required.
